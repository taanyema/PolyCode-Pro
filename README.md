# 🚀 PolyCode Pro Terminal

Un terminal de programmation web mobile-first capable d'exécuter plusieurs langages et de générer des graphiques en temps réel.

## ✨ Fonctionnalités
- **Multi-langage** : Python, C, C++, et Octave (Scilab).
- **Visualisation de données** : Intégration de Chart.js pour des graphiques fluides.
- **Interface Adaptive** : Optimisé pour une utilisation sur smartphone (Termux).
- **Nettoyage Console** : Filtrage automatique des messages système inutiles.

## 🛠️ Installation
```bash
pkg install python octave gcc
pip install flask
python app.py
